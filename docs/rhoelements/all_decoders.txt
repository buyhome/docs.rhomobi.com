##Remarks


###* Exception
Some decoders will not be enabled as some symbologies use the same systems to encode data, making it impossible to differentiate between them. For example: "canpostal" and "auspostal" both use a 4-state system, so devices will only enable either "canpostal" OR "auspostal" when "alldecoders" is enabled.

##Examples

The following example enables all available decoders then turns on the scanner:

	<META HTTP-Equiv="scanner" Content="allDecoders:enabled">
	<META HTTP-Equiv="scanner" Content="enable">
	
The following example enables all available decoders then turns on the scanner:

	<META HTTP-Equiv="scanner" Content="allDecoders:enabled;enable">
